### ExchangeRate-API Details

#### Endpoint
```
GET https://api.exchangerate-api.com/v4/latest/USD
```

#### Response Format
```json
{
  "base": "USD",
  "date": "2026-01-07",
  "time_last_updated": 1736236800,
  "rates": {
    "EUR": 0.9205,
    "GBP": 0.7891,
    "JPY": 149.23,
    "CNY": 7.2456,
    "INR": 83.45,
    // ... 160+ more currencies
  }
}
```

#### Rate Calculation
```javascript
// Example: Convert 1000 USD to EUR
const fromRate = rates["USD"];  // 1
const toRate = rates["EUR"];    // 0.9205
const amountInUSD = 1000 / fromRate;  // 1000
const result = amountInUSD * toRate;   // 920.50
```

#### Error Handling
- Network errors: Show user-friendly message
- API failures: Maintain last known rates
- Invalid responses: Log and alert user
- Timeout: 10 second limit

## üêõ Troubleshooting

### Common Issues

#### Exchange rates not loading
**Symptoms**: "Unable to fetch exchange rates" error
**Solutions**:
1. Check internet connection
2. Verify API is accessible
3. Disable VPN/proxy
4. Try different browser
5. Clear cache and reload

#### Conversion not working
**Symptoms**: Result field stays empty
**Solutions**:
1. Select both currencies
2. Enter valid amount (positive number)
3. Click Convert button
4. Check browser console for errors

#### Dropdown not showing options
**Symptoms**: Empty dropdown list
**Solutions**:
1. Reload the page
2. Clear browser cache
3. Check JavaScript is enabled
4. Try different browser

#### Slow performance
**Symptoms**: Laggy animations, slow response
**Solutions**:
1. Close other browser tabs
2. Update browser
3. Disable browser extensions
4. Check system resources
